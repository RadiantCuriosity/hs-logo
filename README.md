[![Build Status](https://secure.travis-ci.org/RadiantCuriosity/hs-logo.png)](http://travis-ci.org/RadiantCuriosity/hs-logo)

Logo interpreter written in Haskell, specialized for turtle graphics. Still very
much a WIP. Lot of the language still needs to be implemented, but it is fairly
functional already.

More info at http://RadiantCuriosity.github.com/hs-logo

## Quickstart

Follow the [installation instructions](http://RadiantCuriosity.github.com/hs-logo/installation.html)

You can now try out some logo programs in the [examples](https://github.com/RadiantCuriosity/hs-logo/tree/master/site/examples) folder

    % dist/build/hs-logo/hs-logo site/examples/sources/snowflake.logo -o ~/tmp/snowflake.svg

![Snowflake](http://RadiantCuriosity.github.com/hs-logo/examples/svgs/snowflake.svg)

    % dist/build/hs-logo/hs-logo site/examples/sources/design1.logo -o ~/tmp/design1.svg

![Design](http://RadiantCuriosity.github.com/hs-logo/examples/svgs/design1.svg)
